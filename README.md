# Code Documentation

Below is our mapping software used to assist in land claims in the Omusati region, Namibia. For further details on the legal compliance of this software and how it prevents
illicit land claims, refer to the attached manuscript.

## Dependencies / Libraries

The code implements the following dependencies / libraries: 
  - https://unpkg.com/leaflet@1.9.4/dist/leaflet.js
  - https://cdnjs.cloudflare.com/ajax/libs/leaflet.draw/1.0.4/leaflet.draw.js
  - https://cdn.jsdelivr.net/npm/pouchdb@8.0.1/dist/pouchdb.min.js
  - https://cdn.jsdelivr.net/npm/@turf/turf@6/turf.min.js

## Running the code

To run the code on a local http 8000 server:
  1- Run python -m http.server 8000 in terminal
  
  2- Use a browser to search http://localhost:8000/
